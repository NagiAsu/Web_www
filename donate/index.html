# 📌 前言

你好，我是服主 MiunaPA
先感謝各位玩家們，願意遊玩本伺服器
也希望大家可以繼續遊玩及支持我們，謝謝你們

伺服也成立一段時間了，最初伺服開設的想法是提供一個平台，給大家在休閒之餘玩得開心， 而為了提升伺服器遊玩品質，會需要些資金升級硬體，同時平時也會有一些定期支出，以維持伺服的運作，這些成本也是相當大的負擔。

如果你喜歡本伺服器，同時也願意且有能力實際支持我們，歡迎提供贊助幫忙我們，一點小小的肯定可以讓伺服器更好

再次感謝各位玩家們的遊玩及支持！

# 贊助回饋表

  • 贊助100元以上 : 進入主三+設三白名單
  • 贊助200元以上 : 進入主三+設三白名單，另外還可邀請兩位無贊助的玩家加入
  • 贊助300元以上 : 自訂名字顏色(Minecraft 16色)
  • 贊助500元以上 : 自訂名字顏色(RGB顏色)，可使用7個home點，傳送冷卻降低至10秒
  • 贊助1000元以上 : 可使用10個home點，傳送冷卻降低至5秒
  • 贊助2000元以上 : 可使用15個home點，傳送冷卻降低至1秒，可獲得自訂暱稱+第二個自訂名字顏色(RGB顏色)

# 回饋說明

  • 贊助為累積制，過往已贊助的金額會累積上去

  • 進入主三+設三

    • 可直接享有進入主三+設三的白名單
    • 贊助200元以上的玩家，還可再邀請兩位無贊助的玩家進入
      • 請先前往主三或設三，之後在遊戲內輸入指令 : /wl add <要邀請的玩家ID>，來進行白名單的設定
      • 設定後是無法移除的，所以請慎重選擇要加的玩家，唯一例外情況是:若被加的玩家，因贊助獲得白名單後，將會空出邀請名額，屆時可重新進行設定

  • 查看Minecraft 16色顏色代碼 (opens new window)

    • 贊助金額達300元以上，可以從這16色裡面挑一個你喜歡的顏色作為名字顏色

  • 自己RGB配顏色 (opens new window)

    • 贊助金額達500元以上，可以任意調配一個顏色作為你的名字顏色
    • 調整完成後，請使用客服與我們聯繫，將由伺服人員為您進行設定
    • 你可以開一個單人創造，把你想要的顏色改好，打指令自己看會長啥樣子
      /tellraw @s [{"text":"你的ID","color":"#C58181"},{"text":" > ","color":"gray"},{"text":"Hello World","color":"white"}]
      把ID改成自己的 #C58181改成你想要的顏色

  • 自訂暱稱

    • 可以用來做為你的暱稱(在聊天室中取代你的名字顯示)
    • 暱稱設定需透過客服，請在贊助前使用客服與我們討論，以確保您的權益

  • 顏色/暱稱的試用期

    • 顏色及暱稱設定完成後有72小時的試用期(從幫您設定完成後開始算起)，在試用期間，您可以自由提出是否更換目前設定的顏色或暱稱，若超過試用期，往後想要更改顏色或暱稱將酌收200元贊助。
