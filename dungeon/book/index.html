# 魔法書教學

各位同志們好，這邊將教給各位如何操作這個魔術課本第二版。
任何在副本中持有該道具的同志們在使用該道具前都應該先閱讀本操作教學。
在這個操作教學中將學到魔術課本系統第二版的4大組成要素：

  1. 學習
  2. 綁定
  3. 切換
  4. 施放

以上四種方法，各位同志們熟練後可以增加副本的趣味性!

# 學習

就如同該道具的名稱，該道具是一本魔術課本，因此使用前必須先學習變魔術!
在拿到該道具的當下，先拿著往空氣揮一下吧!


這時候就能學會第一個魔術了!
雖然這時候就已經可以直接跳到步驟四的施放了，但是為了以後好還是看下去吧 。
魔術的學習有很多不同的方式，例如從任務取得，所以才有另外的兩個步驟，
再來就往下一個步驟閱讀。

# 綁定

當學習到多種不同的魔術後，為了可以使用這些不同的魔術要先使用附魔台進行
綁定魔術的動作，找到一個附魔台然後拿著魔術課本對著他點右鍵吧。


對附魔台點擊右鍵後會跳出綁定魔術的介面，這時對著想綁定的魔術點擊，
被綁定的魔術會變成一本書的樣子。
再來為了使用這些綁定上去的魔術，要學會怎麼在綁定的魔術中切換，請往下一
步驟閱讀。

# 切換

在綁定完魔術後拿著書往空氣中左鍵揮一下就可以在綁定的魔術中切換不同的魔術
例如我們先前綁定好的魔術有：叉子跟回復，這時我們就可以透過左鍵在兩個魔術
間切換，如下兩張圖：
揮第一下切換成叉子 (CD:10秒)
揮第二下切換成回復

這時候一直揮書本就可以一直切換魔術了。 如果綁定了發現不能切換，可以先重新進副本或是自殺看看。

# 施放

學會上面三個步驟的同志們，現在就可以來變魔術了，利用切換的技巧換成想使用
的魔術後點擊右鍵就可以把魔術變出來了。
每個魔術的冷卻時間不同，如果剛才使用了某個冷卻時間很長的魔術，要注意在冷
卻時間結束前都不能使用其他的魔術，務必小心謹慎。
