    # 新手任務與教學

    第一次進入副本服 你位於的空間是 副本服重生點大廳
    須先完成以下兩項新手教學任務 未來才能接取各項任務 與前往副本服地圖狩獵

    # 1. 新手任務 - [導覽]登記成為冒險者

    首先 請向NPC水果攤大叔接取任務 (右鍵點他)

    接下來 會出現一個介面 請點選以接取該任務

    任務要你去按順序去找以下NPC對話
    (本次任務需要對話的所有NPC 皆位於副本服大廳)

    沙漠商隊的人 位於大廳 櫃台右側

    客家人 位於大廳 回去的門 右側

    邪惡的資本家建商 位於大廳 2樓 (2樓從客家人旁邊的梯子上去)

    櫃台小姐 位於大廳正前方 櫃台 (任務是要你找左側的對話)

    任務完成後 會獲得200元副本幣+1把耐久II鐵劍+10個麵包
    並且未來可向水果攤大叔購買武器

    接下來 請接取下一個任務

    # 2.新手任務 - [新手] 速成課程

    完成後
    向 櫃台小姐(左) 對話 接取任務

    按照任務需求 至櫃台右側的門 進入 鱷魚洞
    打死16隻鱷魚之後 回去與NPC對話 即可完成任務
    獎勵: 鐵劍x1 麵包x16 500元

    # 完成任務後?

    完成以上新手任務後 即可向各NPC接取任務
    並且可至大廳 櫃台後側 的門 進入副本服地圖

    地圖的地形、怪物強度、任務如何解
    這部分是設計給玩家自行摸索的

    前期建議你把每個任務都接一接、地形跑一跑、怪物打一打
    或是跟別人一起解
    久了你就會大概知道怎麼玩了

    # 任務須知

    1.任務流程:
    向NPC接取任務→去打怪→打完之後回去對話→即可完成獎勵
    (需先完成新手教學任務後才可接取任務)
    2.重複接取:
    任務完成後都會有冷卻時間 冷卻結束後可再次接取 部分任務有 任務點數要求
    請輸入 /bs 點擊按鈕查看你目前的任務點數
    3.任務接取: 除了大廳NPC有一些任務外 地圖上也有一些NPC是可以接取任務的
