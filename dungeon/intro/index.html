# 簡介

完全獨立伺服器(不同步資料與背包)
藉由接取任務與打怪、升級裝備獲取副本幣與道具
也是目前賺錢的唯一管道(這是開設副本服的主要目的)

# 基本設定

  1. 死亡不會噴裝(例外)
  2. 不能合成物品
  3. 你的物資=你背包及終界箱內的東西
  4. PVP強制開啟(鱷魚洞除外)
  5. 怪物生成位置基本上不會改變 請自行摸索地圖結構與怪物生成位置
  6. 如果你在地圖卡死了，請輸入/spawn回重生點

# 如何前往副本服

在大廳伺服器 沿著告示牌指示前進

進入此門 即可前往副本伺服器
