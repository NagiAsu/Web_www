# 特殊設定/機制

# PVP懲罰機制

副本的PVP是強制開啟的
但擊殺玩家會有相對應的懲罰機制
以防止惡意影響其他玩家遊玩

# 機制說明

  1. 當你擊殺玩家後，你會進入『PVP警告模式』
    • 期間內再次擊殺玩家，將進入『PVP懲罰模式』
  2. 當進入『PVP懲罰模式』後，期間內會有行動限制
    • 你不能離開副本服，包含上下線/回到大廳
    • 不可使用副本服內的任何傳送門
  3. 在『PVP懲罰模式』期間內違反了行動限制，會受到扣除副本幣的懲罰
    • 若身上已無副本幣可罰款，違反的當下將會立即死亡並噴裝

備註

若你認為機制判定時間有誤請輸入/pvpmode由系統自動除錯

# 抽獎

玩家受到懲罰所扣除的副本幣及物品會進到副本獎池中 提供給玩家抽獎

  • 副本服人數高於一定數量時，會觸發抽獎
    • 獎品為獎池中一定比例的副本幣
  • 每次抽獎會有一定機率觸發追加抽獎
    • 獎品為獎池中一定數量的裝備與物品
