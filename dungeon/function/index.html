# 功能介紹

# 副本幣轉換

將副本幣轉換為一般伺服使用的金錢

# 操作說明

  1. 請輸入/bs點擊此按鈕
  2. 當你看到此訊息後 即可回到主服或設施服
  3. 回來主服或設施服之後 你會看到一條信件訊息
  4. 輸入/mail後 點擊此訊息 即可將信件刪除 並將1000塊領出

# 備註

  • 只有在主服及設施服才能領取金錢

# 副本懸賞系統

如果你希望有玩家去追殺指定玩家，可以懸賞該玩家
任何玩家擊殺被懸賞者一定次數後，將可獲得金錢獎勵

# 操作說明

  1. 至水果攤大叔購買懸賞單(1張200元)
  2. 至2樓懸賞牆，拿著懸賞單對告示牌點右鍵
  3. 選擇想要被懸賞的人
  4. 跳出此通知代表您已完成懸賞

# 備註

  • 只有在副本服內的玩家才可以被懸賞
