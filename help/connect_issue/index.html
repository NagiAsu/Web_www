# 連線問題回報

若您在登入伺服器時遇到了『連線逾時』、『connection timed out no further information』，請參考以下方式處理。

# 確認伺服運作狀況

  1. 請先查看伺服Discord (opens new window)，查看伺服是否有在運作中。
  2. 請查看右側線上列表，正常情況你可以看到機器人存在。
  3. 如果沒有看到，請查看公告頻道，確認伺服是否正在維護。
  4. 也可以看 #公共同步頻道，觀察一下大家是不是有在玩，沒有看出伺服器有異常。

如果你發現伺服運作看起來都正常，但你無法連線，那可能是您的網路有一些狀況，請繼續往下看。
