# 畫作教學

# 安裝模組

# 確認路徑

在安裝模組之前，你需要先找到Minecraft的資料夾才有辦法安裝模組

  1. 打開檔案總管，然後在左側選單找到本機並點擊

  2. 點擊你當初安裝Minecraft的硬碟

  3. 在畫面的上方找到檢視，然後把隱藏的項目勾選起來

  4. 如圖示依序找到.minecraft資料夾，其中Nuko會是你電腦的使用者名稱

  5. 這邊都是Minecraft的遊戲資料，然後點擊mods資料夾，沒有的話就自己新增資料夾

  6. 裡面會有你所有的模組。記住這個路徑，等等安裝會用到

# 下載模組

這邊示範的是基於fabric(設定檔)的litematica(藍圖/投影)模組
依照下列的列表下載模組後，把檔案丟到剛剛的mod資料夾

# 模組列表

這邊不會提供下載連結，請自行將關鍵字貼到google搜尋

  • 必要
    • fabric
      • 設定檔，沒有它你就不用裝了
    • fabric-api
      • 前置模組，沒有它你就不用裝了
    • malilib
      • 前置模組，沒有它你就不用裝了
    • litematica
      • 藍圖/投影模組，不裝它的話就可以關掉這篇教學了
  • 可選
    • OptiFine
      • 可以運行光影包、調整進階顯示設定的設定檔，在此不另外介紹
    • optifabric
      • 讓你能在fabric設定檔中運行Optifine的模組
    • modmune
      • 可查看你所安裝的模組
    • 小地圖
      • 小地圖的種類很多，找自己喜歡的來裝就好了

# 安裝fabric

  1. 點擊你剛剛放進mods資料夾的fabric-installer並執行

  2. 在這個畫面選擇客戶端、Minecraft版本(你要遊玩的版本)、載入程式版本(通常選擇最新即可)

  3. 確認安裝位置是在.minecraft資料夾裡，不是的話照前面的步驟找到並選擇它

  4. 確認沒問題後按下安裝並等待完成即可

# 安裝OptiFine(補充)

  1. 點擊你剛剛放進mods資料夾的OptiFine_1.xx.x_HD_U_Hx並執行

  2. 確認安裝位置是在.minecraft資料夾裡，不是的話照前面的步驟找到並選擇它

  3. 確認沒問題後按下安裝並等待完成即可

# 進入遊戲

  1. 打開Minecraft Launcher

  2. 在設定檔選單中選擇剛剛安裝好的fabric loader，然後開始遊戲

  3. 在單人世界測試模組都能正常運作即完成安裝

# 注意事項

  • 本教學內容僅供參考，必要時請自行利用google搜尋相關資源
    • 大部分模組資源都是以英文撰寫，如果看不懂可以試著丟到google翻譯
  • 強烈建議對電腦操作有一定了解再執行這些操作，避免操作失誤導致電腦無法正常運作
    • 建議事先將.minecraft資料夾完整備份，以方便必要時還原資料
  • 模組下載時請注意版本，強烈建議與你遊玩的版本完全一致，避免發生錯誤
    • 部分情況下可能還是會發生錯誤，屆時請進行交叉測試找出發生問題的模組
      • 有時候選擇較舊版本(模組版本，不是指Minecraft版本)的模組可能會改善此問題
