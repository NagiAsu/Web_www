# 事務說明

# 擺放商人

  • 擺放商人的工本費(不論是新租/轉讓/續租/增加商人) 統一為500元工本費
  • 生物不限制種類，唯一條件是"不能太大隻"(例:凋零/終界龍/地獄幽靈)

# 調整領地負責人 / 店面轉讓

  • 調整領地負責人
    • 就是單純的增加領地負責人，讓他們可以一起幫忙蓋商店
      • 領地負責人不能控制商人及補貨，這只有店長能做
    • 一間店不含店長，最多可有四位領地負責人(1位店長+4位負責人)
  • 店面轉讓
    • 請注意，轉讓是直接更改店長的名字和所有權，同時重置領地負責人
    • 租金期限與位置皆不變
      • 假設今天是7/10，你的店租到7/30，對方會直接繼續用到7/30後才需要續繳下期租金
        中間20天的租金原店主已代繳，這部分由雙方自行協調
    • 轉移本身不用錢
      • 請繼承人之後再填寫問券，並且選擇"擺放商人"的選項，再選擇已繳交500元(放商人)
      • 接著由繼承人繳交500元擺放商人的費用
    • 店面轉讓的申請者(原本持有店面的人)
      • 在申請完之後，請將你的商人全數刪除
  • 玩家ID填寫
    • 如果要增加領地負責人 請填寫
      增加領地負責人 olinger99 LeeTiger125
    • 如果要移除領地負責人 請填寫
      移除領地負責人 LeeTiger125
    • 如果要轉讓店面 請填寫
      店面轉讓 RS1 MiunaPA 轉給 olinger99

# 新商店承租

  • 首先，詳閱商店街須知：https://ppt.cc/fua0nx (opens new window)
  • 再來，去Discord商店街交易頻道及商店列表 (opens new window)確認有無空店面開放承租
  • 確認有店面開放承租後即可填寫商人事務表單 (opens new window)選擇新商店承租，並按照指示填寫表單
    • 現階段請不要繳費，管理員會依照表單送出的時間先後審核，通過後才會在Discord通知繳費
  • 最後，當收到繳費通知時即代表你承租成功，接下來依照管理員指示完成繳費及商人設定即完成承租手續
    • 請務必在管理員給的期限內完成繳費，逾期將會將商店讓給其他玩家承租

支付租金的指令/money pay shop 租金

# 注意事項

  • 請務必按照說明進行準備，以加快同步作業
    • 未按照說明處理畫作有可能會被退件，屆時請重新申請
  • 店面轉讓僅受理一次，若要再次轉讓僅可辦理退租
    • 例：若你的店面是從別的玩家那邊轉讓過來的，要再次轉讓只能申請退租
  • 商店承租後，每個月須定時繳納租金，嚴重逾期者會強制撤租，將空位讓給其他玩家承租
    • 確切日期請依管理員在Discord的通知為主
    • 店面遭強制撤租後會移地保管，屆時須支付罰金才可贖回店面
      • 確切金額請在Discord商店街交易頻道詢問管理員
  • 若伺服方和商店店長解除遊玩契約，店面將會由伺服方強制銷毀
    • 屆時任何玩家都不能承接店面資產(包含領地負責人也是)
  • 申請作業需一定時間，請玩家耐心等候處理
