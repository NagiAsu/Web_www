# 檢舉回報

  • 當你遇到以下的事情時，你可以檢舉回報，請伺服人員幫你處理
    • 我的建築被破壞了
    • 箱子的物品被偷竊了
    • 我的生物被殺了
    • 有玩家嚴重影響你遊玩

處理完成後，你被破壞的方塊/被偷/生物被殺都會還原回來
同時我們會視情況決定是否要與對方解除遊玩契約，確保你的遊玩權益不受影響

請先加入伺服的 Discord，你才能檢舉

# 如何檢舉與示範

檢舉人 : MiunaPA (你自己)
被檢舉人 : kanoooooooooo (破壞你家的人)

  1. 你今天上線時你發現你家被拆了
    你經由查詢發現有玩家破壞你的建築(kanoooooooooo)
    且你要檢舉他(kanoooooooooo)
  2. 請將你使用方塊紀錄查詢
    將查到的紀錄截圖起來(截圖示範在下面)
  3. 截圖必須包含以下資訊
    • 你要檢舉的人(kanoooooooooo)的紀錄
    • 紀錄的座標位置
    • 請盡量清晰可見，不要用手機拍，(下面有截圖教學)
    • 如果你不想讓其他人知道座標，請將此截圖貼到客服
  4. 打開伺服Discord找到檢舉回報
  5. 貼上你剛剛的截圖，並以"文字" 打上伺服器分流/你要檢舉的人的ID/簡述他幹了什麼
  6. 最後，請等待管理員處理
    當管理員處理(解除契約與還原)完成後，你的檢舉回覆會有個表情符號，屆時你再去確認是否有遺漏的地方即可

# 注意事項

# 如果你家被拆得不成人樣(紀錄非常多)

同個範圍內的截圖紀錄貼個幾張就好或是以文字敘述，並且盡量貼上有價值的紀錄

  • 牆壁被拆、玻璃被拆、動物被殺、烽火台被拆、箱子被偷光...
    • 牆壁被拆可以貼
    • 烽火台被破壞紀錄要貼(價值最高)
    • 動物被殺要貼(屬於殺害動物)
    • 偷竊紀錄貼你覺得特別貴重的紀錄(鑽石被偷/附魔裝被偷...)

反正主要重點是，越重要的越要貼，但不用詳細到每筆都貼

# 如果你不想讓別人知道座標

你一樣還是要在Discord檢舉回報頻道，以文字附上伺服器分流/你要檢舉的人的ID/簡述他幹了什麼
並且說明：截圖紀錄在客服

此時開啟客服
將你查詢到的紀錄截圖貼到客服內

# 如果你根本沒有Discord

去辦一個然後加入，不然就是叫你朋友幫你檢舉
