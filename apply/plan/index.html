# 企劃活動申請

# 介紹

我們鼓勵玩家們舉辦任何公開或私人的活動，促進玩家間的交流與互動，並且對本伺服帶來良好的效益。
若玩家有足夠的想法或行動，且需要由伺服方出面協助時，可以與我們聯繫取得相關協助，本篇文將會說明你該如何進行。

若有任何第三方社群或商業合作，建議可直接聯繫本伺服信箱 admin@nagiasu.xyz

# 有什麼是我們可以幫上忙的

  • 臨時的領地劃設，避免場地遭受惡意破壞
  • 臨時傳送點設置，供玩家可以直接性的傳送
  • 管理員的現場協助、監督、主持、參與
  • 活動宣傳
    • 不限定於本伺服或本遊戲，若想協助宣傳第三方遊戲也可協助
  • 特殊道具的申請與提供
    • 以純裝飾性不影響平衡為主
    • 更改物品名字(彩色字)/增加物品敘述/附魔外觀效果
  • 協助提供專用場地或伺服
  • 功能或系統的協助開發與設定使用
  • 任何討論中可能需要，且我們有辦法完成之協助

# 企劃申請方式

請加入伺服Discord，進入『#提問與開發』頻道，接著使用討論串功能建立"公開"討論串，討論串標題請打上你的企劃名稱，建立完成後，在討論串內直接tag服主MiunaPA並等待回應。

# 企劃申請建議提供的相關資料

請務必在企劃資料中詳細說明你的內容，越詳細越好，以方便本伺服能理解
下方的項目僅為舉例內容，實際還是要依照你的想法去撰寫與說明

# 企劃名稱與主旨(必需)

  • 請"簡短"說明本企劃的名稱與主要內容

# 企劃執行時間與地點(必需)

  • 請提供本企劃預計會於何時執行與地點
  • 時間
    • 一個指定時間或一段期間、特殊節慶日子等
  • 地點
    • 若為遊戲內，請附上分流、世界、座標、該地點的名稱(方便後續稱呼)
      • 若該地點為多人居住的社區或他人的建築，請務必事先取得場地所有權人同意
    • 若並非本伺服或本遊戲，請敘述為何種遊戲，並詳細說明該如何前往或參加
      • 也請詳細敘述該遊戲的主要內容

# 報名或參加條件

  • 任何玩家、指定玩家、須滿足某種條件或報名才可參加...
  • 若指定玩家，建議提供完整的參加名單
    • 此名單可不必在討論時先提供(可以先慢慢彙整好)

# 能增進本伺服的效益

  • 請盡可能的敘述說本企劃若能成功執行，對本伺服能帶來的效益
    • 不論是私人或公眾效益都可以，總之盡可能說明

# 企劃對本企劃對全伺服的公平性與合理性

  • 我們希望伺服方對大家都是盡可能公平的，請針對此問題特別思考
    • 你可以公平/不公平/合理/不合理...都沒關係，但請針對此問題，思考與解釋出一個你認為伺服方能接受的答案

# 任何你可以補充的資料

  • 多多益善

# 申請注意事項

# 請在企劃執行時間前14日繳交企劃

  • 請考量到企劃的討論與審核需要時間，若逾期將不受理
  • 企劃方也需保留一定時間，以配合企劃的討論

# 企劃不一定會過

  • 請先抱持這個心情來申請，我們很鼓勵玩家提出企劃，也會盡量去幫忙協助企劃通過與執行，但請考量到伺服方也有一些要求與條件，希望大家能互相理解
  • 伺服方同意企劃後，也代表企劃方對此企劃需負責，而伺服方也需負起同意後的連帶責任

# 企劃的討論過程皆會公開

  • 任何玩家皆可在伺服Discord『#提問與開發』頻道的討論串內找到討論過程

# 沒有企劃就不能申請任何東西

  • 企劃很重要，就算你全部都做好了但沒有企劃，或是企劃申請過後沒有通過，我們是不會幫你的，請依照正規流程
