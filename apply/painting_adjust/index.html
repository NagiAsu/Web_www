# 畫作調整服務

提供玩家針對畫作建築的調整與拆除

  • 畫作蓋錯位置時的搬遷或方向轉換
  • 畫作的免費拆除

# 基本限制

  • 僅提供同分流且同世界內的畫作調整
  • 本服務採客服申請，作業時間全程需1-10天不等

# 收費標準(拆除不用錢)

  • 一張地圖(128x128)大小2500元，單次收費最高上限30000元

# 進行前置作業

# 確認要移動的範圍

  • 決定要調整的範圍，並記下對角線兩點座標
    • 請注意座標不要給錯
    • 可以使用磚頭來右鍵方塊查看位置

# 確認目的地座標

  • 決定移動的目的地，並記下對角線兩點座標
    • 請注意座標不要給錯
    • 目的地的對角線兩點座標的相對位置需與原座標的相對位置一致
      • 這需要一點空間概念，如果你看不懂這句話的話，可以在客服請我們協助您確認

# 確認是否需要旋轉

  • 確認畫作是否需要旋轉，並確認旋轉角度
    • 旋轉角度以90度為單位，並註記順時針或逆時針旋轉
    • 若為原地旋轉，請於目的地座標填入原地旋轉

# 提出申請

  • 以Discord客服提出申請，並照以下格式提供資料 <>填入你的資料

# 畫作的拆除服務

  • 請依照下列圖片所呈現之格式填入地點與座標
  • 第一點與第二點分別填入 x y z 並以空格隔開
  • 若有多張畫要拆除，請分別寫上

# 畫作的移動或調整角度

  • 請依照下列格式填入地點與座標
申請畫作調整服務
<請輸入你的遊戲ID>

分流 : <主一/主二/主三>
世界 : <主世界/地獄/終界>

要移動的範圍
第一點座標 : <x y z>
第二點座標 : <x y z>

目的地座標
第一點座標 : <x y z>
第二點座標 : <x y z>

旋轉方向及角度 (若不需要 這三行不用複製)
方向:<順時針或逆時針>
角度:<以90度為單位>

# 等候移動結果通知

# 移動作業及作業時間

  • 收到案件申請後將由服主進行移動，作業時間需1-10天不等
  • 若在作業過程中發生問題將會立即告知

# 移動結果通知

  • 移動作業完成會馬上通知
  • 收到通知後請盡快到現場確認移動結果
    • 請務必親自確認移動結果以避免造成權益損失

# 確認移動結果

  • 確認移動完的位置、角度
    • 若有任何移動的問題或申請表不同的狀況，請立即提出
    • 若未於2天內確認視為接受移動結果
  • 確定是否需要修正或復原移動
    • 若為玩家的資訊填寫錯誤導致搬遷錯誤(而不是伺服人員的作業錯誤)，有可能需支付額外費用
    • 請於2天內提出申請，逾時需重新進行案件申請

# 支付費用

  • 在以上所有作業完成後，我們會跟你說費用為多少
  • 請玩家支付費用至MiunaPA，並上傳截圖在客服確認
  • /money pay MiunaPA 費用
