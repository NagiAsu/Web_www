# 檢舉回報受理項目的詳細說明

注意事項

受理項目的說明無法涵蓋所有情況，案件是否受理以伺服人員認定之

# 建築被破壞

  • 不限於形式，房子、農場、設施等都可以
  • 自然構造如海底神殿、水晶洞等，需經過改造到能明顯看出有玩家佔領才會受理

# 物品被偷

  • 放在箱子、桶子、展示框裡等等的物品
    • 可使用告示牌上鎖箱子，以及展示框保護套件來保護你的資產
  • 自行丟出去的物品原則上不受理
    • 如果是傳遞物品給玩家時被攔截可以提出檢舉，並附上文字說明

# 生物被殺

  • 你圈養起來的生物
    • 要能明確看出來是被人飼養的生物才會受理
  • 你所飼養的寵物
    • 貓、狗、馬等可辨識主人的生物
    • 例外：跑到公共區域(如商店街、美術館、重生點等)及其周圍時，伺服人員有權直接清除

# 盜印畫作

  • 玩家的地圖畫沒有同步編號
    • 這很有可能是玩家私下盜印畫作，可提出檢舉由伺服人員協助查證

# 被其他玩家影響遊玩

  • PVP關閉時被岩漿、打火石燒；被TNT、終界水晶炸等
    • PVP開啟時一律不受理
  • 副本服僅受理外掛相關案件，其餘包含搶怪、殺人皆不受理
  • 其他會影響遊玩的行為

# 有玩家做出違反遊玩規範的行為

  • 使用外掛/利用遊戲BUG、開分身/使用VPN等
