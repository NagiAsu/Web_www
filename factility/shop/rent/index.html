# 租金優惠

  • 在繳滿6個月後，將可獲得1個月的租金退還!
    • 形同於第六個月是免費的 (-16%)
    • 店主需準時繳納租金，若有遲交過久且未事先告知情況者，將不適用此優惠。
    • 採事後認列，店主仍須每個月按時繳納租金，伺服方會另外退還租金給你。
    • 若中途該店面有更換店主，新店主將重新計算累積期數。
      • 若更換店主時原店主"同時"承租了新的店面(更換店位搬家)，原店主的期數可繼續累積。
      • 但必須是要退租的同時又承租一間，才可視為店位搬家，若是退租後一陣子再承租則重新累積。
    • 單月租金<=2500以下的店面不適用此優惠。
    • 本優惠是長期計畫，若租滿12、18....個月後也將享有此優惠。
