# 伺服 Discord

  • 一個很好用的通訊和聊天軟體，電腦與手機皆支援
  • 註冊帳號 https://discord.com/ (opens new window)

# 加入的邀請連結

# https://discord.gg/E33Mk8y (opens new window)

# 加入方式

# 如果你已經登入伺服了 (建議)

  • 請進行身分綁定，綁定完成即會獲得身分組
  • 請參考下方的『連結綁定說明』

# 如果你還沒登入過伺服

  • 請在新人大廳輸入你的遊戲ID
  • 完成後，請靜候伺服人員審核通過即可加入
  • 這會花比較久，建議先登入伺服，並進行綁定身分

# 連結綁定說明

讓你的Discord帳號，與遊戲內角色連結起來

# 使用方式

  1. 首先，請先登入遊戲
    同時要加入 伺服Discord
  2. 在遊戲內輸入指令 /link

    你會得到一串六位數字
  3. 進入伺服Discord，找到下面『🎯身分驗證』
  4. 在該頻道內直接輸入六位數字
  5. 當出現連結成功後，即完成綁定
  6. 你可以確認是否擁有『玩家、已連結』這兩個身分組

# 若已綁定

如果你已經綁定過，但沒有『玩家、已連結』這兩個身分組
(可能是因為你退出過群組，造成身分組消失，或是系統錯誤導致沒給到身分組)
可在『🎯身分驗證』(不是遊戲內) 輸入/link
系統會幫你將身分組加回來

# 若要解除綁定

如果你換過遊戲帳號或Discord帳號，或是個人因素等，需要解除綁定
請使用 客服 協助處理
