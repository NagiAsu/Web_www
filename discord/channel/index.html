# Discord各頻道說明

排序方式由上至下

# 聊天

# 💬聊天室

  • 不限主題聊天，平時的聊天請多加善用這裡

# 🦖遊戲專區

  • 適合用來聊其他遊戲(不屬於本伺服)的地方

# Minecraft伺服器

# 💻同步頻道

  • 此頻道的訊息會與伺服內聊天室雙向同步
  • 需綁定遊戲帳號才可在此頻道發言連結綁定
  • 此頻道的所有發言，皆視為在遊戲內說話，請遵守伺服規定

# 📢公告

  • 伺服的更新/維護/活動...資訊皆會於此公告
  • 請玩家密切關注此頻道的訊息，以確保了解伺服的更新資訊

# MC事務

# 🚔檢舉回報

  • 若要檢舉玩家，請使用此頻道
  • 檢舉方式與格式請參考檢舉破壞
  • 非相關人士請勿在此聊天，若要調解也請自行建立討論串或至玩家調解區

# 🧬提問與開發

  • 若遊玩時有遇到任何問題，可在此發問(也可使用客服系統)
  • 有任何對伺服的想法或意見也可在此提出，鼓勵玩家多多參與
  • 請不要出現"伺服內部無關的事情/與伺服無關的問題/無意義且無法幫助的回答"

# 🏪商店街交易

  • 有關商店街的資訊會於此宣布
  • 商店列表 : https://shop.nagiasu.xyz/ (opens new window)
  • 有關商店的事務申請請參考事務申請

# 🎨伺服圖分享

  • 歡迎大家在這邊貼出在本伺服遊玩的圖片
    • 請勿張貼非本伺服的圖片
  • 請盡量不要在這邊聊天

# ⛳活動企劃

  • 若伺服有舉辦任何活動需要報名或公告活動資訊，會在此頻道公告

# 語音區

  • 歡迎玩家自由使用語音頻道
  • 同時，若想使用Discord Bot指令(例如點歌)，也請在語音區內輸入

# 公眾與說明

# 🎯身分驗證

  • 連結伺服角色與Discord帳號
  • 詳細說明請參考連結綁定說明

# 🌍公開聊天室

  • 提供給尚未綁定的玩家聊天的地方

# 事務處理

  • 包含『💵換錢處理』、『🎪商店事務』、『🌈畫作同步』
  • 如果你有申請任何事務處理，處理的結果會在這邊公告，可以在這查看
  • 當我們處理完成後，會上一個表情代表處理完成
