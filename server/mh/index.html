# 設施服 mh

建造紅石機關/生怪塔/物資塔/大型紅石設備

  • 如果有需要蓋上述很吃效能的東西，請在此伺服建造
  • 不是很建議蓋家居住在此伺服，建議前往主伺服(sv)
  • 若需採集資源，建議前往資源服(res)

  • 使用/home或/back傳送到位於該分流的紀錄點

# 分流設定

# 基本設定

  • 地圖防爆、防燒
  • 防噴裝、死亡扣除部分經驗
    • 死亡時扣除5等
      • 玩家經驗等級在10等以下時不扣除經驗

# 特殊設定

  • 有指定掉落物定時清除機制
    • 指定掉落物：竹子、甘蔗、西瓜、南瓜、海帶、仙人掌
  • 終界使者不會撿起方塊
  • 綿羊不會受到雷擊傷害
    • 仍然建議玩家在羊毛機附近設置避雷針
  • 無法繁殖綿羊
    • 可使用綿羊卷軸提升單隻綿羊的產量
  • 無法召喚終界龍
    • 請前往資源服(res)
  • 不會生成夜魅
    • 請前往資源服(res)
  • 不會生成任何魚類
    • 請前往資源服(res)
  • 伏聆觸媒不會蔓延
    • 請前往資源服(res)
  • 無法挖掘地獄的遠古遺骸
    • 請前往資源服(res)
    • 請不要在地獄放遠古遺骸，挖掘時會被系統清除
  • 原木剝皮/銅塊除蠟需使用有絲綢之觸附魔的鑽石/獄髓斧
    • 資源服(res)無此限制
    • 單純對銅塊除鏽可使用任意的斧頭

# 切換世界/分流方式

# 切換世界

  • 前往地獄請自行找地方蓋地獄門
  • 前往終界請輸入/spawn回到分流重生點後由傳送門前往

# 切換分流

  • 前往大廳服(Lobby)
    • 輸入bs後點選指南針前往
  • 前往對應的主伺服(sv)(主一/設一 主二/設二 主三/設三)
    • 輸入/spawn回到分流重生點後由傳送門前往
  • 前往其他分流 商店街 美術館
    • 請切換至大廳服(Lobby)後由各分流傳送門前往
