# 主伺服 sv

玩家建築蓋家的地方

  • 禁止蓋生怪塔及任何耗費大量實體或資源的設施
    • 如有此需要，請前往設施服(mh)
  • 若需採集資源，建議前往資源服(res)

# 前往方式

  • 從大廳服(Lobby)的各分流傳送門前往
  • 使用/home或/back傳送到位於該分流的紀錄點

# 分流設定

# 基本設定

  • 地圖防爆、防燒
  • 防噴裝、死亡扣除部分經驗
    • 死亡時扣除5等
      • 玩家經驗等級在10等以下時不扣除經驗

# 特殊設定

  • 關閉生怪磚功能
  • 生怪效率調降
  • 村民不會被殭屍攻擊/感染
  • 終界使者不會撿起方塊
  • 無法正常觸發突襲
    • 需使用1450呼叫器
    • 身上帶有不祥之兆效果進入村莊範圍會被系統隱藏
      • 隱藏時仍會計算效果時間
      • 離開村莊範圍後效果會正常顯示
      • 效果隱藏時請不要切換分流或登出，效果會被系統清除
  • 無法召喚終界龍
    • 請前往資源服(res)
  • 不會生成任何魚類
    • 請前往資源服(res)
  • 不會生成夜魅
    • 請前往資源服(res)
  • 部分生物有繁殖限制
    • 牛、豬：10格內小於10隻才可繁殖
  • 無法繁殖悅靈
    • 請使用/bs選單購買生怪蛋
  • 無法繁殖村民
    • 請前往設施服(mh)
  • 無法繁殖界伏蚌
    • 請前往設施服(mh)
  • 無法使用發射器剪羊毛
    • 請前往設施服(mh)
  • 伏聆觸媒不會蔓延
    • 請前往資源服(res)
  • 無法挖掘地獄的遠古遺骸
    • 請前往資源服(res)
    • 請不要在地獄放遠古遺骸，挖掘時會被系統清除
  • 原木剝皮/銅塊除蠟需使用有絲綢之觸附魔的鑽石/獄髓斧
    • 資源服(res)無此限制
    • 單純對銅塊除鏽可使用任意的斧頭

# 切換世界/分流方式

# 切換世界

  • 前往地獄請自行找地方蓋地獄門
  • 前往終界請輸入/spawn回到分流重生點後由傳送門前往

# 切換分流

  • 前往大廳服(Lobby)
    • 輸入bs後點選指南針前往
  • 前往對應的設施服(mh)(主一/設一 主二/設二 主三/設三)
    • 輸入/spawn回到分流重生點後由傳送門前往
    • 輸入bs後點選左上角按鈕前往
  • 前往其他分流 商店街 美術館
    • 請切換至大廳服(Lobby)後由各分流傳送門前往
