# 重生點/spawn

大廳服的重生點，離各分流傳送門最近的地方

正面直行：往主伺服傳送門

背面直行：往資源服、玩家商店街、美術館傳送門

往副本服的動線，請詳見副本服 簡介

# 前往方式

  • 在各分流輸入bs後點選指南針前往
    • 副本服請於重生點點選NPC前往
