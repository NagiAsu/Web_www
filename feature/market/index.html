# 市場資訊說明

市場資訊：該品項在市場上的基本資訊
單位數量：每交易一個單位所代表的物品數量
價格區間：上架收購/販售單時允許設定的價格區間
販售倉量：目前市場上有多少物品正在販售，其中個人及全伺服有個別的上限
目前收購市價：目前將每單位物品賣給收購市場會獲得的金錢，也是目前收購單中最高的價錢
目前販售市價：目前從販售市場購買每單位物品會獲得的金錢，也是目前販售單中最低的價錢
七日交易資訊：過去七天該品項在市場上的交易狀況，可用以判斷市場當前的熱度及行情
總成交量：該品項在收購市場及販售市場上的總成交量
成交價格區間：該品項在收購市場及販售市場上成交的價格區間
平均成交價格：該品項在收購市場及販售市場上的平均成交價格

# 判斷熱度及行情

# 市場熱度：

可從七日總成交量得知目前指定物品的交易熱度，越高代表參與買賣的人越多。
而熱度越高不一定代表能更快從販售/收購中得到收益，要同時參考販售倉量，倉量越高通常越慢。
如果想要從市場上得到更多的收益，可以試著考慮交易熱度相對低一些，但販售價格也不差的商品。

# 行情價：

最直接的方法是參考七日平均成交價格，將它當作目前的市場行情價。
而為了得到更具參考性的資訊，通常我們會加上總成交量以及成交價格區間綜合評估，這需要一點統計的概念。 有時候也可以參考玩家在商店街或是私下販售的價格，這可能會比市場熱度極差時的平均成交價格值得參考。

# 購買商品的優先順序

  • 先買最低價格的商品(由最低價往上購買)
    • 想更快賣掉商品得到收益的話可以適度地降低販售價格，幅度越大通常越快
  • 相同價格則以最久的上架時間來購買(先進先出)

# 販售商品的優先順序

  • 先賣最高價格的商品(由最高價往下購買)
    • 想更快買到商品的話可以適度地提高收購價格，幅度越大通常越快
  • 相同價格則以最久的上架時間來購買(先進先出)

# 收購單上架

  • 上架方式 : 輸入/buy <市場名稱> <你想要收購的每單位價格> <單位>
    • EX: 假設你想收30單位沙子 你希望每單位收5.0元
    • 輸入/buy sand 5.0 30
  • 上架需先支付保證金與手續費! 每單會向你收2%的上架費用 最低0.1元
    • EX: 收購 沙子5.0元x30單位保證金150元 上架手續費為3元
  • 若要撤銷收購單，請點至沙子的商品頁面，並點擊綠色界伏盒來撤銷收購單(會退還你剩餘的保證金)
  • /purchaseshelf t查看你目前收購的物品

# 販售單上架

  • 上架方式 : 輸入/sell <市場名稱> <你想要販售的每單位價格> <單位>
    • EX: 假設你身上有30單位白色羊毛 你希望賣每單位5元
    • 輸入/sell white_wool 5.0 30
  • 上架有手續費!每單會先向你收5%的上架費用 最低0.1元
    • EX: 上架 白色羊毛3.0元x20單位上架手續費為3元
    • EX: 上架 白色羊毛2.0元x1單位上架手續費為0.1元
  • 若要撤銷販售單，請點至商品頁面，並點擊紅色界伏盒來撤銷收購單(會退還你剩餘商品)
    • 撤銷販售單需另外支付2%撤銷手續費
  • /sellshelf t查看你目前販售的物品

# 領取販售單收益

登入時如果有販售單有賣出東西收益 會跳通知告訴你
賣家輸入/sellreceive即可領錢

# 領取收購單貨物

登入時如果收購單有收到東西 會跳通知告訴你
買家輸入/stock即可領取收購到的商品
