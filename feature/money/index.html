# 金錢系統

伺服自製的金錢系統，可以用來跟玩家交易

注意事項

副本服的帳戶與其他分流獨立，無法直接轉帳，詳細請參考副本服 功能介紹

# 指令

  • /money help查看幫助選單
  • /money查詢自己有多少錢
  • /money <玩家ID>查詢別人有多少錢
  • /money pay <玩家ID> <金額>給別人錢
    • 指令發送後會有提示文字及提示音，須等待兩秒後才可按下確認轉帳
  • /money top [起始名次]查看金錢排行榜
    • 未輸入會顯示 前1~15名
    • 輸入起始名次會顯示 前n~(n+15)名
  • /money lookup <頁數>查看交易紀錄
    • 每10筆一頁
    • 當你給別人錢的時候，若對方不在線上或與你不同分流，對方不會收到轉帳通知，此時你可以通知對方請對方輸入此指令確認是否有收到錢
    • 橘色字的尾數是交易編號，主要用於商人繳費的表單填寫
