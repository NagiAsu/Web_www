# 畫作系統

# 買家用指令

  • /maplist <玩家ID>查詢指定玩家的畫作
    • 輸入這個指令可以查詢指定登記在該玩家名下的所有畫作，點擊價格可以購買指定畫作
    • 其中每幅畫的格式皆為：名稱 張數 價格 (ID) 展示
      名稱：該幅畫作的名稱
      張數：該幅畫作的張數大小(有幾張地圖)
      價格：該畫作的價格，顯示未設定可能代表絕版或非公開販售，有興趣請自行詢問作者
      ID：該畫作的ID
      展示：該畫作有沒有公開展示，綠色可點擊預覽；紅色為R18畫作，可前往美術館預覽；
      　　非公開則沒有公開展示
  • /mapbuy <畫作ID>購買指定畫作
    • 若你知道特定畫作的ID，輸入這個指令可以購買指定畫作

便利功能

若你身上有空的界伏盒(沒有顏色的)，且畫作大於9張時，會優先放在盒子裡

# 作者用指令

  • /maplist setprice <畫作ID> <價格>設定畫作價格

    • 輸入這個指令可以設定自己名下指定畫作的價格

  • /maplist setprice <畫作ID> clear清除畫作價格

    • 清除價格以將畫作設定為未設定

  • /mapreceive領取畫作收益

    • 輸入這個指令可以領取畫作售出的收益，其中會扣除每張地圖5元的成本

  • /mapconvert畫作轉換

    • 當畫作同步完成後，可以手持未同步的地圖輸入該指令轉換成同步的畫作
      其中手持的地圖需與申請同步的地圖相同ID才能成功轉換

# 建立畫作告示牌

建立跟美術館一樣，可以右鍵購買畫作的告示牌
設定方式：放置告示牌，第一行輸入[畫作購買]，第二行輸入畫作ID，第三行輸入額外想打的字(可選)
