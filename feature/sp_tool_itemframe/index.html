# 功能性道具 - 展示框保護套件 / 隱形展示框

# 說明

可以用來保護你的展示框，使他如同鎖住的箱子一樣，不會被其他人摧毀

  • 其他人無法旋轉與打下你在展示框上的物品，也無法打掉展示框
  • 任何怪物/爆炸/弓箭...皆無法對展示框造成影響
  • 若被方塊擠壓，或是懸浮在空中，也不會掉下來

# 取得方式

  • /bs選單 -> 交易市場 -> 雜物
  • 售價：每16個 200元 + 1獄髓碇 + 1鑽石磚
  • 一次性道具，使用後消失

# 使用方式

  • 首先，請先將展示框與要展示的物品放在展示框上
    並且手持 展示框保護套件 對該展示框點擊"左鍵"
  • 這樣就保護完成了

# 注意事項

  • 保護後的展示框，如果你打掉(取下)該物品，展示框會直接連同物品掉落。

# 隱形展示框

  • 隱形條件：展示框必須先保護，才能進行隱形
  • 使用方式：手持金色胡蘿蔔，對你保護的展示框點擊"左鍵"，即可將之隱形
