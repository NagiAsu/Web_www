# 鎖箱子/門

將箱子/門鎖定後，即可限制只有誰才能打開箱子/門(本伺服沒有領地)
請玩家盡量將儲存私人物品的容器進行鎖定

# 使用方法

對著箱子/門放告示牌即可鎖定完成

如果只是想放告示牌在箱子/門上，但不想鎖定箱子/門
請先蹲下 再放置告示牌

# 可鎖定物件

  • 箱子類、熔爐類、發射器容器類、漏斗、釀造台 (界伏盒不能鎖)
  • 門/柵欄門/地板門 也可以鎖定

# 增加使用人

# /bl <行號> <使用者的ID>

  1. 右鍵點擊告示牌
  2. 輸入/bl <行號> <使用者的ID>即可編輯
    (告示牌有四行 從上往下數 2 3 4)

# 增加更多牌子

若ID不夠位置放可以在同個箱子/門上放第二個告示牌

(圖片有點大 要等他跑一下)

# 讓所有人都能使用

  • 可以讓箱子/門設定成每個人都能開跟拿東西
    但只有你能拆掉箱子/門
  • 適合用在公共箱子/建築

# 增加標籤 [所有人]或[Everyone]

  • 輸入/bl <行號> [所有人]
    • EX : /bl 3 [所有人]

# 定時關門 [Timer:時長]

  • 輸入/bl <行號> [Timer:時長]
    • EX : /bl 3 [Timer:1]

# 允許用漏斗傳輸物品

  • 鎖定箱子後，物品依然能用漏斗/漏斗礦車流進箱子裡
  • 希望你鎖定的容器可以利用漏斗『流出物品』
  • 此設定只有主伺服才需要(設施服/資源服不用設定)

# 增加標籤 [紅石]

  • 輸入/bl <行號> [紅石]
    • EX : /bl 3 [紅石]

# 備註

  • 現在並不支援顏色字設定
  • 早期的鎖箱牌子上的特殊顏色會因為使用而消失，這是正常的
