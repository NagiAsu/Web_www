# 物品署名

# 功能介紹

  • 署名後的物品，上面會有你的名字
  • 只有署名玩家才能解除署名
  • 只有署名玩家才能使用鐵砧改名、修復
    • 非署名玩家使用鐵砧會無效，且經驗值及物品仍會消耗
  • 可以將你的工具之類的上署名，就能知道這是誰的了
  • 地圖不能署名，請使用畫作同步系統

# 加上/解除署名

  • /signature add將物品加上署名
    • 手持物品(數量不限，一次拿一堆就是全部署名上去)
  • /signature remove將物品解除署名
    • 只有當事人能解除署名

# 署名樣式

# 署名發行

  • /signature release將物品署名發行
    • 手持由你署名的物品(數量不限，一次拿一堆就是全部發行)
    • 建議使用方式
      1. 先將物品進行署名
      2. 將署名的物品以鐵砧改名
      3. 再使用此功能 能讓改名後的物品名稱變成正體字
        (不然你拿鐵砧改會是斜體)
    • 可用在物品的驗證
      • 只有你做的出來署名&正體字的道具 別人做不出來
        用以確保這一定是你的
      • 例如 : 自己做抽獎券/兌換券...

# 注意事項

  • 方塊使用後(放置、合成等)署名會消失
  • 署名是紀錄ID 所以如果你要改ID 請務必先解除署名
    • 當你更改ID後若需要解除署名，請利用客服系統請管理員協助處理
