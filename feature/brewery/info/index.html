# 基礎教學

# 基本的釀造方法

  • 首先 你需要一個 釀酒鍋

  • 釀酒鍋製造方法：一個鍋釜裝水 並在下方擺上營火

  1. 裝一桶水倒進鍋裡
  2. 手持原料 並右鍵鍋子 即可加入鍋裡
    (右鍵1次會加入1個原料 要自己記得加了哪些原料跟數量)
    並開始等待
  3. 等待時間可手持時鐘右鍵 查看烹煮時間
  4. 當你覺得時間OK了 用3個空玻璃瓶撈起來 會獲得半成品/成品
  5. 接著依照酒類需求 決定是否繼續 蒸餾或貯存

# 蒸餾

許多酒類煮完後只會得到半成品
需要進行蒸餾繼續加工

# 蒸餾台製造方法

釀造台 裡面放一個螢光粉
(螢光粉不會被消耗)

# 蒸餾次數

當釀造台次數跑完1次
代表已完成蒸餾了1次(飲品上會寫)
不同的酒有不同的蒸餾次數

# 貯存

有些酒可以儲藏起來繼續加工
此時你需要建造酒桶來放

# 存放木桶

  • 製造方法：酒桶 長4寬3高4
  • 建材 : 木材x18，木樓梯x16，木柵欄x5
  • 在前方貼一個木牌寫上Barrel
    並提示貯藏桶創建成功才可用。
    (有不同木種的存放木桶，各個都有不同的酒類能存放)

# 貯存時間

  • 遊戲時間1天=酒放了1年
  • 酒不是越放越久越好

# 釀造相關知識

  • 飲品有品質高低，所加入的材料以及燉煮的時間會影響成品的品質，
    同時也決定飲品是否成功。
  • 釀造參數，包含
    • 材料數量
    • 燉煮時間
    • 蒸餾次數
    • 貯存木桶種類
    • 貯存年份
  • 將飲品放入釀造台可以得知那項飲品的酒精濃度，
    把飲品放入存放木桶一小段時間則可以看其品質。
  • 每種酒都有屬於他最完美的參數
    酒譜只會給出範圍，參數越靠近該值，酒的品質越高(影響飲品的效果強度)
  • 難度(1-10) 為酒能接受的精準度
    難度越高的酒 參數與精準值差異越大 則越容易失敗

# 酒醉

  • 喝下酒之後會酒醉，走路也會搖搖晃晃，可喝東西或吃東西來解酒
    • 酒醉效果會跨分流
  • /brew info查看自身酒醉程度。
  • 解酒物品
    • 麵包 -2
    • 牛奶桶 -5
    • 熱牛奶 -20
    • 金蘋果 -20

# 調酒

  1. 將調酒需要的材料 放入鍋中 煮2分鐘後取出 此為調酒成品
  2. 取出的調酒基底 放入蒸餾器蒸餾一次 即可完成
  3. 通常使用不好又不差的酒

# 釀造配方

  • 這邊只會給釀造的變數範圍
    最完美的值請自行摸索出來
  • 沒有寫蒸餾 代表該飲品不需要蒸餾
  • 沒有寫貯存 代表該飲品不需要貯存放木桶
  • 要貯存的酒一定會跟你說要貯存
    但不會給你年份範圍

# 特殊材料

  • 有些酒需要使用特殊材料
  • 調酒產生的特殊材料 會以 斜體 表示
  • 酒 會以 粗體 表示
  • 由副本入手的材料 會在材料前加一個#

# 密封

酒在製成之後，可以將其密封，密封完成的酒將無法繼續貯存年份與蒸餾
也不能查看其年份、材料、蒸餾次數(只會顯示幾星)

在你製作好酒之後很適合密封起來放/賣給別人

  1. 首先 需要合成一個密封槽 (外觀是煙燻爐)
    橡木材x4+空瓶子x2
  2. 將其放置在地上
    右鍵點開 將要密封的酒
    放在九宮格上任意位置
  3. 等待約1秒後 即完成密封 以下是密封前後
