# 特殊道具

一些正常生存模式下不會出現的特殊道具

  • 用於因應伺服特殊設定的功能性道具
  • 針對玩家特殊需求的輔助型道具

# 功能性道具

# 比石頭還硬的磚頭

  • 用途：自殺用，用於回到重生點/床點
  • 注意事項：冷卻時間120秒
  • 取得方式：副本任務[石頭與磚頭] (opens new window)
  • 關懷資源：全國自殺防治中心: https://www.tsos.org.tw/ (opens new window)
    　　　　　自殺防治專線: 1995
    　　　　　請愛惜生命。

# 狐狸包巾

  • 用途：用於改變狐狸的毛色(在紅狐狸與雪狐狸之間變換)
  • 注意事項：一次性道具，使用後消失
  • 取得方式：/bs選單->交易市場->生怪蛋->可信任生物
  • 售價：500元

# 蘿莉濃湯

  • 用途：用於讓幼年生物維持在幼年狀態(不適用於小殭屍)
  • 注意事項：一次性道具，使用後消失
    　　　　　請不要使用在其他玩家或自己身上
  • 取得方式：/bs選單->交易市場->生怪蛋->其他生物
  • 售價：500元

# 1450呼叫器

  • 用途：用於在主伺服(sv)觸發突襲，使用時須拿在主手上
  • 注意事項：一次性道具，使用後消失
    使用時會記錄玩家及座標，請不要拿來影響其他玩家
  • 取得方式：/bs選單->交易市場->雜物
  • 售價：1450元

# 綿羊卷軸

  • 用途：用於提升單隻綿羊的羊毛產量，建議在設施服(mh)使用
  • 注意事項：一次性道具，使用後消失；每次皆為獨立事件，沒有保底
  • 機制說明：每次使用依照機率提升綿羊等級(機率與提升等級依照卷軸上標示)
    　　　　　最高提升等級：設施服(mh)：+20級/主伺服(sv)：+1級
    　　　　　羊毛掉落公式(使用剪刀/發射器)：1~3個(原版)+0~3個x每個等級(獨立事件)
    　　　　　理論值：每個等級會提升產量1.5個，理論上+20級的羊每次會產出32個羊毛(長期平均)
  • 取得方式：副本任務[一卷軸都是毛任務] (opens new window)(70%機率+1等級)
    　　　　　伺服官方活動獎勵(不定時舉辦；機率與提升等級不一)

# 展示框保護套件

  • 用途：可以用來保護你的展示框，使他如同鎖住的箱子一樣，不會被其他人摧毀
  • 注意事項：一次性道具，使用後消失
  • 取得方式：/bs選單->交易市場->雜物
  • 售價：每16個 200元 + 1獄髓碇 + 1鑽石磚
  • 詳細使用說明請點我

# 扳手

  • 用途：用於編輯盔甲架的外觀(大小、角度、基座等)，使用時須拿在主手上
  • 注意事項：手持此道具時無法開啟容器/攻擊/破壞
    　　　　　可搭配命名牌鎖定盔甲架，防止他人竄改(但不防止射箭及縱火的破壞)
    　　　　　若玩家更改ID，請自行射箭移除盔甲架
  • 取得方式：副本任務[配給搞丟了?有沒有空?可以來拯救嗎?] (opens new window)

# 輔助型道具

# 土木仔的秘製生財工具

  • 用途：提升挖掘深板岩的速度，加上挖掘加速二效果可以達成瞬間破壞
  • 道具數值：效率8，耐久9，無法附魔
  • 注意事項：消耗性道具，僅可破壞深板岩
  • 取得方式：副本任務[關於讀書人的專業] (opens new window)
  • 紀念品：耐久耗盡後會掉落秘製生財工具的木柄

# 土木仔的秘製生財工具 改

  • 用途：提升挖掘指定方塊的速度，部分方塊加上挖掘加速二效果可以達成瞬間破壞
  • 道具數值：效率8，耐久14，無法附魔
  • 注意事項：消耗性道具，僅可破壞指定方塊
  • 取得方式：即將推出，敬請期待(細節可能會有所更動，以實際推出為主)
