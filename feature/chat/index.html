# 聊天室

# 密語指令

用來發送密語給別人，可以跨伺服傳送

  • /tell <玩家ID> <文字>發送密語給他人(/m , /msg 也是同樣效果)
  • /r <文字>密語回覆上一個發送你密語的人

# 禁言指令

用來禁言玩家，阻擋你不想要看到的訊息
禁言範圍包含公頻(遊戲內及DC同步訊息)及密語

  • /ignore <玩家ID>將指定玩家禁言
    • 重新輸入即可解除禁言
  • /ignore查看被你禁言的玩家
    • 會在聊天欄列出列表

# 頻道系統

用來管理接收/發送訊息的指令

  • /c開啟頻道列表

    • 用來管理接收哪些頻道的訊息

      • 移動到按鍵上可查看說明
  • /ch chat <頻道代號>設定發話頻道
    • 頻道代號可在指令/c的介面中 將滑鼠移到頻道列表上查詢
    • 部分同步頻道不開放遊戲內發送訊息，詳見Discord文字頻道

# 其他功能

  • 在遊戲內tag玩家(對方須綁定Discord)
    • 使用方式：@<玩家ID>
  • 在遊戲內傳送伺服Discord表情符號
    • 使用方式：:<表情符號ID>:
  • 調整上下線訊息顯示方式
    • 使用方式：/lm
  • 將訊息傳送至Discord客服系統
    • 使用方式：/mcs <訊息內容>
    • 須將遊戲帳號綁定Discord帳號
    • 如尚未建立客服頻道會立即開啟
    • 訊息會在遊戲內及Discord中同步

# 遊戲內寄信

  • 在遊戲內寄信給其他玩家，對方會在上線或切換分流時收到信件
    • 使用方式：/mail send <玩家ID> <內容>

# 展示物品[item]

用來在聊天室展示自己身上的物品
[]前後需與其他文字空格

  • 主手：[item] [i] [inv] [hand]
  • 副手：[offhand](若打兩次展示主手的文字 後面的會自動轉換為副手)
  • 頭盔：[helmet] [head] [hat]
  • 胸甲：[chestplate] [body]
  • 褲子：[Leggings] [Legging] [leg]
  • 鞋子：[Boots] [boot]

# 名稱系統

用來管理遊戲內稱號、名字顏色及暱稱顯示

  • /tg開啟名稱系統介面
    • 稱號獲得方式請待日後公告
    • 名字顏色及暱稱為贊助回饋
      • 移動到按鍵上可查看說明
  • /tguse使用稱號兌換卷
    • 主手持稱號兌換卷輸入指令即可兌換
