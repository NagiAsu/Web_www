# Discord 客服系統使用方式

# Discord

  1. 首先 請先加入 伺服Discord
  2. 加入後 請在左側頻道列表 往下滑動至最底層 點選 客服系統

    動畫圖片操作說明 (opens new window)
  3. 請點擊按鈕 點我開啟客服

  4. 完成後 這時候下方就會出現一個頻道了
  5. 請在此頻道提出你想詢問的問題/申請/等...並靜候伺服人員回覆

# 遊戲內

  • 輸入/msc <訊息>將訊息發送至客服系統(若客服頻道不存在會立即建立)
    • 訊息會在遊戲及Discord中同步(如果你有綁定Discord)

# 注意事項

  1. 請不要濫用此功能
  2. 不要用客服問管理員一些跟伺服完全沒關係的問題
  3. 客服頻道會在使用完成後刪除頻道 請自行保存紀錄
  4. 伺服方保有將客服紀錄公開的權利
