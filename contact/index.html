# Discord 客服系統

  1. 請加入伺服Discord https://discord.gg/nagiasu (opens new window)
  2. 請在左側頻道列表往下滑動至最底並點選客服系統(操作說明動畫 (opens new window))
  3. 請點擊按鈕點我開啟客服

  4. 完成後左側選單最下方就會出現一個你專屬的客服頻道了
  5. 請在此頻道提出你想詢問的問題/要申請的事務等，並等候伺服人員回覆訊息
  • 備註：關於客服系統的詳細說明請點我
