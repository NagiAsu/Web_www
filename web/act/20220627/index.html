# 風平浪靜 NagiAsu.xyz 建築比賽 2022/06/27

企劃主辦人 : neko_vanilla (香草)

# 比賽內容

  • 每位參賽者會以生存模式，在範圍為30X30的平地裡(基底為平原草地)，生態域為一般的平原，並以官方提供的材料在限時內進行比賽
  • 材料為主辦方提供，每人都一樣，比賽開始後開箱子才可得知材料內容，用完不補
  • 場地大致上是長這樣 30x30，高度不限，旁邊有材料箱子
  • 題材與風格完全由你自由發揮，我們只限制你建材只能使用我們提供給您的
  • 完成後，請為你自己的建築訂題目與自行想要怎麼說明，箱子內有一本書，請寫在裡面
    • 題目:1-16字內
    • 說明:1-100字內
      • 字數長度不會直接影響你的分數，所以不用要求自己寫滿，以你認為最有辦法介紹你的題目與建築為主。

# 比賽設定

  • 玩家:生存模式，不可PVP，可自由合成物品，不會飢餓
  • 世界設定
    • 生態域 : 草地平原 plain
    • 有日夜與天氣循環
    • 和平模式
  • 伺服
    • 會提供專用活動伺服，從大廳進入，不同步背包

# 比賽時間

  • 報名時間:
    • 活動開始前10分鐘皆可報名
  • 比賽時間(可建築時間)
    • 報到時間 : 2022/06/27(一) 19：00 ~ 19：15
      • 報到完成後，就會立即給你生存權限及建造權，即可直接開始構思建造
      • 如果趕不上也可以晚點報到，但這就會壓縮到你的參加時間
    • 2022/06/27(一) 19：00 ~ 21：15
    • 2022/06/28(二) 12：00 ~ 19：00
      • 開始時間可能會視情況調整，但保證至少會有連續六小時
    • 建造時間會保證給你至少2+6小時的可建造時間，但該用多久或何時進來建造則取決於你
    • 參賽者隨時可自由加入與離開活動場地，但開放時間就只有上述時段，請自行控制時間分配與進度

# 比賽規則

  • 禁止抄襲他人建築
  • 請勿搗亂(成立者一律失去比賽資格)
  • 不可違反伺服與公頻規範，不可有限制級題材
  • 禁止使用藍圖模組與自動放置

    • 評審認可為完成品的，即可獲得活動紀念品
    • 參考圖片如下
  • 參加獎
    • 2000遊戲幣
    • 極少部分情況，若評審認為這是來亂的或認為完全0分，將不會獲得參加獎

# 參加資格

  • 加入官方Discord綁定超過七天，與遊玩本伺服超過七天者，即可在Discord活動頻道報名。

# 評分方式

  • 由各位評審評分，每位評審會需要參考以下表格來進行評分，最後將各評審之評分加總，依照總分來比較名次。
    • 請尊重各個評審的評分，參賽者不得對評審提出分數異議。
    • 評審名單會在比賽開始前確定人數與名單，屆時會公告

# 評分標準如下

  • 美觀 30%
    • 看上去舒服，裝飾美觀
  • 結構性 20%
    • 結構的合理性與協調性
  • 環境表現 20%
    • 地質背影的改造使大局觀更美麗
  • 創意程度 20%
    • 天馬行空的幻想及材料的特殊運用
  • 作品介紹文 10%
    • 介紹你的作品並敘述你的工法及特色
  • 整體比例 10%
    • 以Minecraft人物為準
  • 細緻表現 20%
    • 對於你建築上的細部處理方式，主要在接近觀看時的內容表現
  • 彩蛋表現 10%
    • 彩蛋內容注意不得影響善良風俗及社會觀感
  • 相似度 10%
    • 你的題目是否呼應你所呈現的作品
