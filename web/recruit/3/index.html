# 招募說明

這次就長話短說了，由於伺服未來的規畫調整，我們需要招募新任的管理員，若您有擔任的意願，且認為自己有能力的玩家，歡迎加入我們。

加入前，請務必先詳細閱讀上一期的說明二期招募文件 (opens new window)，裡面有詳細記載伺服人員的條件與要求，本篇文件僅說明調整的部分。

# 招募細項

本次主要招募的是"管理員"，負責平時的事務處理與主動巡邏的工作，詳細工作與說明請參考二期招募文件 (opens new window)。
另外，我們希望這次的管理員，能時常且主動願意上線工作。

# 加入方式

  • 首先，請加入招募群組(寫在公告上)，加入群組後的應徵方式，請看該群組公告
    • 之後將下面大格子的問題複製起來，在問題下面 將<你的回答>換成你所要回答的答案
    • 最好能多寫點你的想法
  • 接著，我們會與你在該應徵頻道討論，了解雙方的想法
    • 請盡量不要讓我們等你的答案等太久，建議開啟所有通知
    • 私人頻道內的討論與你的回答，我們都不會公開(但會有紀錄)， 全程只有現任管理員知道，請放心回答
  • 最後會進行公告招募結果

# 問題說明

  • 基本資料
    • Minecraft ID
      • 請輸入你的遊戲ID
    • Discord ID
      • 必須包含#後面的四個數字
    • 你的年齡
      • 請不要謊報
    • 你目前的事情
      • 如果你目前是學生，請回答你的學位階級(以今年九月開始計算 國三/高一/大四...)
      • 如果你不是學生，則請填你目前做的事情(休學/上班(不用寫職業)/待業...)
    • 平時是否可使用語音?
      • 是/否/可以但不能隨時用...簡略敘述即可
  • 遊戲理解
    • 玩Minecraft多久
      • 你玩這遊戲多久了，回答幾年幾月之類的即可
    • 玩這個服多久
      • 同上，回答幾年幾月之類的即可
    • 有玩過其他伺服嗎
      • 有的話，可以說一下你玩過什麼服(名字)，如果這裡是你玩的第一個多人伺服(沒玩過其他伺服)，回答無即可
    • 能簡單描述一下那個服在玩什麼嗎
      • 伺服的類型(RPG/小遊戲/原味生存/模組...)，若上題回答是無，則本題一樣回答無即可
  • 經驗
    • 有沒有在其他伺服當過任何伺服人員職位的經驗?
      • 有的話，請簡述之前的工作內容與想法，沒有則直接寫無就好
    • 對於人際溝通與團隊合作參與上，你能舉一些個人經驗或溝通等經驗嗎?
      • 任何只要你覺得你曾經參與過，且需要上述能力的事情與經驗，都可以自由列舉和舉例
      • 例如學校:系會長/社長 團體:企劃參與 工作:PM/主管 遊戲:公會長/隊長 社群:群主...
  • 時間
    • 若你上任後，你的時間規劃是什麼
      • 上任的時間大約是六月中左右，能簡單說明一下你現實有什麼事情要做嗎?這關係到能否時常上線幫忙的部分，我們也不希望一直要求管理員要現在上線幫忙，進而影響到你的現實生活事務。
  • 評量
    • 你認為你自己在一般玩家之間的形象為何?
      • 基本上這是個自評而已，以你自己所認為的想法去回答即可
  • 其他
    • 任何你覺得你可以補充的加分條件
