# 遊玩規範

風平浪靜 NagiAsu.xyz，提供Minecraft多人伺服器遊玩、Discord社群、網站、資訊、服務等，供使用者遊玩與使用。
當您加入伺服器遊玩後，即視為您已詳細了解並同意本規定；若您對本規定有任何疑慮，應立即向伺服方詢問。 若不願意遵守本規定，遊玩方應立即停止使用本服務。

  • 基本上，在路上看到人造建築與箱子不要任意去亂碰
