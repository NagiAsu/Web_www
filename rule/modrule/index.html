# 嚴禁使用外掛功能/遊戲BUG 詳細說明

# 可使用不具外掛功能的輔助型模組與客戶端

  • 一般模組
    • 小地圖、整理背包、光影
    • 藍圖(Litematica)、連點
      • 不可用於活動或使用腳本運行
      • 不可使用printing功能
    • tweakeroo
      • 不可用於活動或使用腳本運行
      • 不得做出等同於外掛的行為，由伺服方認定之
  • 客戶端
    • LunarClient、Badlion

注意

未明載之模組或客戶端若要使用，強烈建議事先詢問客服是否允許

# 嚴禁使用任何具外掛功能的材質包/模組/客戶端

  • 任何具外掛或透視功能的材質包/模組
  • 使用含有外掛功能之客戶端遊玩 (不論是否開啟外掛功能)
  • 飛行、自動建設、自動戰鬥、KillAura...
  • 包含但不限於未明載且具外掛功能之材質包/模組/客戶端，由伺服方認定之

# 利用遊戲BUG或從中獲取利益

  • 原版BUG : TNT、鐵軌、地毯複製機...
  • 伺服BUG：伺服異常、插件漏洞
  • 認定標準 : 『遊戲機制』和『實際玩法』不一致時，以遊戲本身的設計用意為主
