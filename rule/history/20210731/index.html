# 🔧 伺服規定 2021-07-31

  • 請勿影響別人遊玩(例如：破壞建築、偷東西、騷擾玩家...)
    • 基本上，在路上看到人造建築與箱子不要任意去亂碰
  • 嚴禁任何外掛行為
    • 如果有使用一些客觀上有疑慮是否外掛的模組 請務必先以客服詢問，否則將依照伺服人員認定之處分
    • 以下幾種可使用： 小地圖、整理背包、光影、藍圖(Litematica)[活動時禁止使用] 其餘模組若要使用 請詢問客服
  • 請勿使用遊戲 bug
    • 例如:TNT、鐵軌、地毯複製機...
  • 請勿開分身或使用VPN遊玩
    • 有此需求請一律由客服詢問或申請，否則將依照伺服人員認定處分
  • 嚴禁盜取或複製他人的畫作
    • 違規者視情況給予處分
  1. 伺服處分以帳號與IP為準，請務必保管好個人帳號 (切勿外借給任何人，若發生任何狀況須自行承擔責任) 不接受任何因個人因素導致之理由作為要求解除處分之依據
  2. 當你發現有玩家有違反以上規定時 請將狀況回報至Discord，交由管理員處理 (切勿知情不報)
  3. 若對於規定有任何疑慮的部分 請務必先詢問伺服人員，否則將依照伺服人員的認定方式處理
