# 伺服規定 2021-10-24

  • # 請勿影響其他玩家遊玩或伺服運作

    • 破壞建築、偷東西、騷擾、詐欺、惡意誤導、無意義LAG機...等影響他人的行為。
      • 基本上，在路上看到人造建築與箱子不要任意去亂碰
    • 嚴禁以非正當管道取得畫作
      • 非正當管道係指未經作者同意逕自盜印或複製畫作
    • 嚴禁有任何影響伺服運作之行為或預謀
      • 包含但不限於以網路或電磁紀錄等方式影響伺服運作
      • 為確保伺服運作的穩定，此類個案的處置依據將可能溯及既往
      • 必要時，伺服人員需向全體玩家公開判決依據及理由
      • 若有嚴重危害而致違法疑慮者，將不排除採取法律途徑
    • 若有上述之情事發生，伺服人員擁有主動介入處置之權利
      • 未明載但屬本項規定所涵蓋之事項，將由伺服人員認定及處置

  • # 嚴禁使用任何外掛與遊戲BUG

    • 以下幾種模組可使用：
      • 小地圖、整理背包、光影
      • 藍圖(Litematica)、連點
        • 不可用於活動或使用腳本運行
      • 其餘未明載之模組若要使用，建議事先詢問客服是否允許使用
    • 以下模組或功能嚴禁使用(包括但不限於)
      • 任何具外掛或透視功能的材質包/模組
      • 使用外掛客戶端遊玩(不論是否開啟外掛功能)
      • 飛行、自動建設、自動戰鬥...
    • 遊戲BUG
      • 原版BUG : TNT、鐵軌、地毯複製機...
    • 伺服異常或插件漏洞
      • 利用或從中獲取利益
    • 若玩家知悉上述之情事發生，請即時利用客服通報，切勿知情不報
      • 知情不報者經伺服人員查獲時將一併進行處置
    • 若有上述之情事發生，伺服人員擁有主動介入處置之權利
      • 未明載但屬本項規定所涵蓋之事項，將由伺服人員認定及處置

  • # 請勿開分身或使用VPN遊玩

    • 若有這類需求，請務必主動利用客服向伺服人員詢問或提出申請
    • 違反本項規定將依照伺服人員之認定進行處置

# 處置與上訴

  • 違反上述伺服規定，伺服有權利對玩家進行以下的處置
    • 暫停或終止玩家的遊玩權限
    • 暫時限制玩家在遊戲聊天室發言的權利
    • 刪除或回溯玩家的遊玩紀錄
  • 玩家權利上訴
    • 玩家可在處分確定後由客服詢問/調解/上訴
      • 上訴期限 : 判決確定後一個月內
  • 處分資料
    • 資料係指:處分依據與解釋判決
      • 帶有個人隱私之內容需消去處理
        (例如:IP / 座標 / 私人檢舉...)
    • 當事人有權利詢問與要求伺服人員提供資料
      而伺服人員有義務向當事人提供與解釋
    • 雙方皆有權利向全體玩家公開資料以接受公評

# 備註

  1. 伺服處分以帳號與IP為準，請務必保管好個人帳號(切勿外借給任何人，若發生任何狀況須自行承擔責任)，不接受任何因個人因素導致之理由作為要求解除處分之依據。
  2. 若發現有玩家有違反以上規定時，請即時利用客服，切勿知情不報。
  3. 若對於規定有任何疑慮或意見，請利用客服主動詢問伺服人員。
  4. 本規定適用範圍 : 風平浪靜 Minecraft Java 伺服器。
