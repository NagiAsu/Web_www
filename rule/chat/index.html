# 公眾頻道聊天規範

各位玩家好，這裡是個公開的伺服環境，您的每一句發言都會影響社群的風氣，為了維護伺服的遊玩體驗、營造良好的環境，讓玩家們都可以在此伺服開心的遊玩，因此希望各位玩家們遵守並多加配合!

同時頻道的發言也會同步至伺服Discord
也請遵守 : Discord社群守則 (opens new window)

# 請勿發送有關謾罵、歧視、性騷擾、不雅字眼、嚴重冒犯他人的相關言詞或影音

  • 當在此頻道發言時，請務必考慮到你的每一句話會被全體玩家所看見。
  • 發言時請顧及並尊重不同玩家與族群的感受，避免造成他人不適。
  • 相關言詞之認定包含但不限於諧音、注音文等。
  • 請勿發表違反善良風俗或有違法之虞的言詞或影音。

# 請勿洗頻、傳送惡意釣魚連結、散播個人資料

  • 此類情況會嚴重影響其他玩家的遊玩體驗，我們會加重處理。
  • 此類情況之認定不限於文字或影音形式。
  • 洗頻之認定不限於一人或多人參與。

# 請勿惡意騷擾玩家

  • 包含上述要點行為，若接收方認為不舒服的，可截圖記錄並檢舉。
  • 此類狀況包含遊戲或Discord的私人訊息之騷擾。

# 請不要在此頻道與他人爭執或發生衝突

  • 不論爭執事件的是非對錯，請考慮到在此頻道上的其他玩家會被動且強迫的看見你們的爭吵內容，這對於與其他無關的玩家是一則沒有必要的訊息，亦或是一種干擾。
    • 不論爭執的參與者是事件的當事人、當事人朋友、無關的旁觀者...。
    • 包含當某玩家的行為在你眼中認為是不當行為，進而公開表達個人立場或情緒並當眾指責。
    • 當該事件與您無關時，請盡量不要加入衝突的參與(這可能反而會助長衝突的擴大)，而是優先聯繫管理員來出面處理。
  • 若與玩家發生衝突，還請在其他非此頻道處理，或是使用客服系統請管理員出面協助。
  • 嚴禁有引戰等助長衝突之行為發生或意圖。

# 請盡量避免將此頻道當作自己的個人群組或私人聊天室

  • 對於其他完全不認識你或是無法參與你的話題的玩家來說，這可能會是一則無法理解且不必要的訊息。
  • 若為互相熟識之玩家，鼓勵至非此頻道聊天，這樣在聊天時會比較自由，也不會對其他玩家造成干擾。
  • 同時，也請盡量不要進行廣告之行為。

# 內容認定

  • 由您的帳號(包含Minecraft帳號/Discord帳號)所發言之任何內容與資訊載體，皆視為由您本人之發言。

  • 包括但不限於 : 文字、物品、多媒體、貼圖、連結、暱稱、帳號#數字。

  • 任何非公開範圍，伺服皆無法得知與保存任何內容

    • 包含 : 社區頻道/遊戲或Discord私訊/社區私人Discord...。
    • 伺服任何人包括服主自己，皆不會也無法得知其聊天內容為何，請玩家們放心自由聊天，相關記錄也請自行保存。
    • 鼓勵玩家多多使用私訊/社區頻道或第三方平台，享受完全由玩家自治的聊天環境。
