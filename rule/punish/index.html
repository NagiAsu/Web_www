# 終止遊玩契約

  • 違反上述伺服規範，伺服人員有權利取消提供下列服務給玩家
    • 暫時限制玩家在遊戲聊天室發言的權利
    • 暫時或永久限制用戶在Discord發言/加入的權利
    • 暫時或永久停止玩家在本伺服的遊玩權限
    • 刪除或回溯玩家在本伺服的遊玩紀錄
    • 刪除違反規定之相關發言紀錄
  • 因應案件處理之必要，伺服人員得做出下列臨時處置
    • 暫時以臨時領地保護案件現場
    • 暫時停止玩家在本伺服的遊玩權限，並要求回覆客服訊息
  • 未明載但屬伺服規範所涵蓋之事項，將由伺服人員認定及處置
  • 若玩家知悉上述之情事發生，請即時利用客服系統通報，切勿知情不報
    • 知情不報者經伺服人員查獲時將一併進行處理

# 案件調解

  • 主要適用於『玩家與玩家之間的檢舉』
  • 主要由被檢舉人親自向檢舉人提出，伺服方不會主動介入
    • 被檢舉人也可由客服告知伺服人員，請伺服人員作為第三方協助處理
  • 調解期限 : 檢舉人提出檢舉後，檢舉『還原完成』前
    • 若在還原完成後才提出調解，需由伺服人員同意才可受理
  • 調解流程
    • 若玩家在檢舉後，被檢舉人主動聯繫檢舉人相關調解事宜，可暫緩檢舉處理
      • 檢舉人有拒絕調解的權力
    • 請檢舉人在檢舉頻道告知伺服人員此檢舉在調解中
    • 由雙方自行私下調解(或由伺服人員協助雙方)相關補償或處理方式
    • 若調解成功，該檢舉將可撤銷，視為此案件未曾發生過

# 案件上訴

  • 玩家權利上訴
    • 玩家可在判決確定後由客服詢問/調解/上訴
      • 上訴期限 : 原則上為判決確定後一個月內
  • 解約原因所依據的資料
    • 資料係指:解約依據與解釋判決
      • 帶有個人隱私之內容需消去處理
        (例如:IP / 座標 / 私人檢舉...)
    • 當事人有權利詢問與要求伺服人員提供資料
      而伺服人員有義務回覆並向當事人提供與解釋
    • 雙方皆有權利向全體玩家公開資料以接受公評
